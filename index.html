Apa Itu Termux?

Termux adalah aplikasi terminal emulator yang menyediakan lingkungan Linux lengkap di Android. Dengan ini pengguna bisa menjalankan command Linux, install package, menjalankan script, dan belajar sistem operasi langsung dari HP tanpa komputer.


Termux mendukung banyak tools seperti Python, Git, Node.js, PHP, SSH, dan berbagai utilities Linux lainnya. Aplikasi ini sangat cocok untuk belajar programming, networking, dan system administration.


Dengan Termux, HP kamu bisa berubah jadi mini Linux server yang bisa digunakan untuk belajar, eksperimen, dan pengembangan project.


Fungsi & Kelebihan

  • Menjalankan Linux command di Android
  • Install tools developer (Python, Git, Node.js)
  • Bisa bikin web server lokal
  • Support SSH remote server
  • Bisa install distro Linux (Ubuntu, Debian, dll)
  • Gratis dan open-source
